The complete browser

Features

Everything Minova 1.0 brings together, from the address bar to protected media playback, with the controls kept close and the interface kept calm.

BrowsingPersonalizationPrivacyExtensionsMediaPerformance

Everyday browsing

The familiar workflows people rely on, built into Minova's own focused desktop shell.

Real Chromium pages

Modern Chromium rendering, site isolation, permissions, downloads, and full web compatibility through Castlabs Electron.

Smart omnibox

Navigate directly or search from one field, with suggestions from recent pages and previous searches as you type.

Session restore

Continue with regular tabs after restarting. Private tabs are intentionally never written into the saved session.

Tabs that work like tabs

Open, close, pin, restore, reorder through normal workflows, and close the final tab to exit the browser.

History and bookmarks

Browse recent history and saved bookmarks directly from the three-dot menu or dedicated management pages.

Find, print, and downloads

Native find-in-page results, printing, download tracking, and quick access to completed files.

Personalization

Change the interface at a deeper level than a single accent switch.

Four theme modes

Follow the system, choose light or dark, or unlock a fully custom theme.

Nine independent colors

Control page background, panels, raised controls, borders, primary and secondary text, two accents, and warnings.

Live preview

See color changes immediately in tabs, controls, settings, menus, and internal pages.

Custom New Tab shortcuts

Add your own destinations to the New Tab page alongside the built-in quick links.

Search engine choice

Use Google, Bing, DuckDuckGo, Brave, or define your own search URL.

Home and startup controls

Choose a home page and decide whether Minova restores your session, opens New Tab, or starts somewhere specific.

Privacy and passwords

Useful protections that stay understandable and close to the browsing experience.

Built-in ad and tracker blocking

Filter common advertising and tracking requests with locally cached, regularly refreshed block lists.

Encrypted local vault

Credentials are protected using Electron safeStorage backed by Windows encryption, never written to a plaintext password file.

Account suggestions

Saved logins for the current site appear where account forms need them, with autofill controls in Settings.

Private tabs

Browse in an isolated private session that does not enter Minova's saved history or session restore.

Granular site permissions

Control notifications, camera, microphone, location, pop-ups, and Do Not Track behavior.

Clear data controls

Delete browsing data on demand, clear all history at once, or erase browser data automatically when Minova closes.

Extension compatibility

A practical Chrome-extension layer with clear expectations about Electron compatibility.

Manifest V2 and V3

Load compatible extensions with common runtime, storage, tabs, windows, permissions, cookies, and context-menu APIs.

Chrome Web Store installer

Paste a Web Store URL or extension ID and let Minova fetch and install the extension package.

Load unpacked

Developers can select a local folder containing a manifest and test an extension directly.

Pinned extension actions

Pin compatible extension icons beside the address bar and open their popups without leaving the current page.

Media and streaming

Keep video useful beyond the tab, including services that require a certified protected-media path.

Integrated Streaming Mode

Protected services can use the installed Microsoft-signed Edge playback path while Minova's tabs and toolbar remain attached.

Streaming service detection

Minova recognizes major services and can suggest Streaming Mode when direct protected playback is unavailable.

Always-on-top video

Pop compatible media into a compact window that stays visible over games and other applications.

Built-in volume boost

Raise quiet media beyond its normal page volume from Minova's toolbar controls.

Media-aware tabs

Tab state tracks active audio and video so media remains visible and is not suspended unexpectedly.

Site favicons

Tabs display each site's real icon, making YouTube, Netflix, Steam, and everyday pages easy to distinguish.

Performance and maintenance

Minova saves resources without treating every background tab the same.

Smart tab suspension

Choose standard or custom timeouts and wake suspended tabs without losing their URL or position.

Activity safeguards

Downloads, audible media, video, capture, fullscreen pages, unsaved forms, uploads, and pinned tabs are protected.

Per-site exclusions

Keep important domains awake with a simple website exclusion list.

Silent background updates

Minova checks GitHub releases, downloads updates quietly, and lets the user install now or wait until normal exit.

Manual update checks

Check from the three-dot menu or About Minova and see clear checking, downloading, ready, and up-to-date states.

Open development

GPL-licensed source, repeatable installer builds, release metadata, checksums, and automated browser regression tests.
