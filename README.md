# Minova Chromium

Public website and release home for Minova Chromium, a customizable Chromium
desktop browser for Windows.

- [Product website](https://minova-chromium.github.io/Minova-Chromium/)
- [Feature catalog](https://minova-chromium.github.io/Minova-Chromium/features.html)
- [Bug reports and suggestions](https://minova-chromium.github.io/Minova-Chromium/feedback.html)
- [Latest release](https://github.com/minova-chromium/Minova-Chromium/releases/latest)

The website is static HTML, CSS, and JavaScript, with no build step. Feedback is
submitted from the site to `minova.chromium@gmail.com`.
